# CRM for Travel Agencies

The solution is built as a full-stack app with:
- ASP.NET Core (.NET 10) backend
- React + TypeScript frontend
- SQL Server persistence

Main scope includes customer management, travel programs/departures, bookings, payments, and controlled document metadata handling.
